Sample data

# Generate some example data for clustering
tmp <- c(rnorm(30,-3), rnorm(30,3))
x <- cbind(x=tmp, y=rev(tmp))
plot(x, main = "sample cluster data")

Using the kmeans() on our sample data

k <- kmeans(x, centers = 2, nstart = 20)
